Google Finnance


Spreadsheet workbook dedicated to logging and monitoring an investment portfolio.
Transactions data are entered to the tables in the appropriated sheets.
ROI, P/L, NAV are automatically calculated using real time data provided by google finance.
Historical data is used to compare the portfolio to the performance of the S&P500 index.

Trending Movies


The 10 most trending 4K/x265/HDR movies listed in the public torrent tracker rarbg.to
The raw data is gathered using their dedicated API, cleaned, trimmed and joined by metadata from The Open Movie Database.
Updated daily via GitHub actions.
Check out the source code in the GitHub repo.
